Hi, This is a Virtual Assistant Program for Desktop, which allows you to communicate with you PC without even touchning it.


HOW TO USE:
---------------------------------------
There is no installation process
1. Click on the executable file.
2. Wait for the program to run.
3. Click on the Mic button to speak.
4. Say Exit to leave.
